2. 安装并配置php

3. 安装并配置apache2

通过apt-get命令安装apache2 sudo apt-get install apache2

修改apache2配置文件vi /etc/apache2/apache2.conf

完成后启动\etc\init.d\apache2 start

启动apache2服务后测试配置是否正确，在浏览器中输入本机ip

如果显示welcome apache界面，说明apache2安装配置成功。
